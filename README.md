# 飲食店共有ルーム

- グループ内で飲食店の情報を共有できるアプリです
- 「部屋」というグループを作り、そのメンバーにだけ情報が公開されます
- レスポンシブ対応しています

URL: https://restaurants-we-know.herokuapp.com/

## 使用技術

- Ruby 2.7.1
- Ruby on Rails 6.0.6
- MySQL
- Docker
- Heroku
- Cloudinary

## 機能一覧

- ユーザー登録、ログイン機能(devise)
- 部屋メンバー申請機能
- 飲食店情報登録
- 飲食店感想登録
- 写真投稿機能(carrierWave)
